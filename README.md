# Espejo Real · Tu piel en 1 minuto

Archivos listos para GitHub Pages.

## Qué subir
Sube todos estos archivos a la raíz del repositorio:

- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

## Activar GitHub Pages
Settings → Pages → Branch: main → /root → Save.

## Importante
Después de subir cambios de PWA, prueba en ventana de incógnito o borra caché del navegador.
En Android puede aparecer botón de instalar.
En iPhone normalmente se instala desde Compartir → Añadir a pantalla de inicio.
